In deze opdracht moeten we kijken uit welke css code het grootste font zal komen:

  1. H1 met als css voor de grootte: 2ex
  2. H1 met als css voor de grootte: 2em

De het antwoord is nummer 2, omdat het em element de grootte van het font gebruikt en die in dit geval maal 2 doet(in % gezien 200%)

het ex element staat voor de x-hoogte van het font die je gebruikt(in dit geval dus als x-hoogte: 2

Dus de grootste zal dus nummer 2 zijn omdat deze het font dubbel zo groot maakt
