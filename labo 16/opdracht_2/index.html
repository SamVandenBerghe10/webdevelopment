op de html site zelf wordt er een klasse "slider" aangemaakt bij de slider, dit element wordt hierna in de javascript code in een variabele gestoken: "sliders", hierna wordt er met dit element een eventlister gebruikt om te kijken wanneer de slider van waarde verranderd en ten slotte zullen deze eventlisteners de methode "update" oproepen die naar de console zal printen

In 2016 hebben sommige browsers verranderd hoe hun sliders werkt waardoor "input" en "change" nodig zijn om voor alles compatibel te zijn
